# Cursor Skills 集合

自用的 Cursor Skills 集合，用于 Cursor IDE 中的 AI 写作与发布流程。

## 目录说明

- **写作**：通用写作流水线（入口与约束 → 选题 → 素材收集与清洗 → 证据分级与受众扫描 → 主张与支撑点 → 通用骨架与段落计划 → 目的覆盖器 → 起草与母稿质检 → 平台适配 → 发布与反馈），含主调度与 01～10 步子 Skill。
- **发布端**：发布相关 Skill。
- **复盘端**：复盘相关 Skill。

## 使用方式

将本仓库中的文件夹（如 `写作`、`发布端`、`复盘端`）复制到项目的 `.cursor/` 目录下，或按 Cursor 规则配置的 skills 路径放置，即可在 Cursor 中通过技能名称或路径调用。
